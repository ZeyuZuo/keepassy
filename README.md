# KeePassY

A modern, cross-platform KeePass desktop client backed by a high-performance Rust core.

## Overview

KeePassY is a **KeePass (.kdbx) password manager** for desktop use. It combines a **Flutter** frontend with a **Rust** backend to deliver a responsive, feature-rich user experience while keeping your secrets safe.

| Layer | Technology | Location |
|-------|-----------|----------|
| UI | Flutter / Dart | `keepassy_flutter/` |
| Business Logic | Rust | `keepass-rs/crates/keepass_core/` |
| C ABI Bridge | Rust | `keepass-rs/crates/keepass_ffi/` |
| Debug CLI | Rust | `keepass-rs/crates/keepass_cli/` |

## Features

- **Open local `.kdbx` files** — password-only, keyfile-only, or password+keyfile
- **WebDAV support** — open and save remote vaults with ETag-based conflict detection
- **Create new databases** — create a new empty KeePass vault from within the app
- **Full entry management** — create, edit, delete, duplicate, and move entries across groups
- **Custom fields & protected values** — add, edit, and delete custom string fields with protect toggle
- **Attachments** — view, add, replace, export, and remove entry attachments
- **Entry history** — browse and inspect read-only historical snapshots
- **Groups** — create, rename, delete, and restore groups with full Recycle Bin support
- **Recycle Bin** — recoverable deletes with original-group restore
- **Search & sort** — global search across all entries and sort controls
- **Password generator** — configurable password generation (length, character sets, avoid ambiguous)
- **Change master password** — re-encrypt the database with new credentials
- **Auto-lock** — configurable inactivity lock timer
- **Clipboard clearing** — automatic clipboard clearing after a configurable timeout
- **Theme support** — system, light, and dark themes
- **Keyfile support** — keyfile-based database unlock and creation

## Architecture

```text
Flutter UI  →  Dart Repository  →  Rust FFI  →  keepass_core  →  Storage
    │               │                  │              │
    │               │                  │        KeePass parsing,
    │               │         C ABI adapter      validation, save,
    │        VaultRepository        │         WebDAV, history,
    │         (abstract)            │         attachments, etc.
    │                               │
  Material 3                 libkeepass_ffi.so
```

Key principles:
- Widgets depend on `VaultRepository`, never call FFI directly
- `keepass_core` stays UI-independent and testable
- `keepass_ffi` is a thin C ABI adapter over `keepass_core`
- Save is explicit — mutations mark dirty but never auto-persist
- Cross-boundary data uses JSON DTOs

## Prerequisites

- **Rust** 1.80+ (`rustup` recommended)
- **Flutter** SDK with Linux desktop support (see [Flutter desktop setup](https://docs.flutter.dev/desktop))
- **Linux** (primary target; Windows/macOS support is planned)

## Quick Start

### 1. Build the Rust shared library

```bash
cd keepass-rs
cargo build -p keepass_ffi --release
```

The shared library will be at `keepass-rs/target/release/libkeepass_ffi.so`.

### 2. Run the Flutter app

```bash
cd keepassy_flutter

# Point to the built FFI library
export KEEPASSY_FFI_LIB="../keepass-rs/target/release/libkeepass_ffi.so"

flutter run -d linux
```

### 3. Release build

```bash
scripts/build_linux_release.sh
```

Output: `dist/linux/KeePassY/`

# KeePassY

一个基于 Flutter 和 Rust 构建的现代化 KeePass 桌面客户端

## 概述

KeePassY 是一款面向桌面的 **KeePass (.kdbx) 密码管理器**。它采用 **Flutter** 前端配合 **Rust** 后端的架构，在保障密码安全的同时提供流畅、功能丰富的用户体验。

| 层级 | 技术 | 位置 |
|------|------|------|
| 用户界面 | Flutter / Dart | `keepassy_flutter/` |
| 业务逻辑 | Rust | `keepass-rs/crates/keepass_core/` |
| C ABI 桥接层 | Rust | `keepass-rs/crates/keepass_ffi/` |
| 调试命令行 | Rust | `keepass-rs/crates/keepass_cli/` |

## 功能特性

- **打开本地 `.kdbx` 文件** — 支持仅密码、仅密钥文件、密码+密钥文件三种方式
- **WebDAV 远程同步** — 通过 WebDAV 打开和保存远程保险库，支持 ETag 冲突检测
- **创建新数据库** — 在应用内直接创建新的空 KeePass 保险库
- **完整的条目管理** — 跨群组创建、编辑、删除、复制和移动条目
- **自定义字段与保护值** — 添加、编辑、删除自定义字段，支持保护开关
- **附件管理** — 查看、添加、替换、导出和删除条目附件
- **条目历史** — 浏览和查看只读的历史快照
- **群组管理** — 创建、重命名、删除和恢复群组，支持完整的回收站功能
- **回收站** — 可恢复的删除操作，支持恢复到原群组
- **搜索与排序** — 全局跨条目搜索和排序控制
- **密码生成器** — 可配置的密码生成（长度、字符集、排除歧义字符）
- **修改主密码** — 使用新凭据重新加密数据库
- **自动锁定** — 可配置的无操作自动锁定计时器
- **剪贴板清理** — 超时后自动清除剪贴板内容
- **主题支持** — 跟随系统、浅色和深色主题
- **密钥文件支持** — 支持使用密钥文件解锁和创建数据库

## 架构

```text
Flutter UI  →  Dart Repository  →  Rust FFI  →  keepass_core  →  Storage
    │               │                  │              │
    │               │                  │        KeePass 解析、
    │               │         C ABI 适配层      验证、保存、
    │        VaultRepository        │        WebDAV、历史记录、
    │         (抽象接口)              │         附件管理 等
    │                               │
  Material 3                 libkeepass_ffi.so
```

核心原则：
- Widget 层依赖 `VaultRepository` 抽象接口，绝不直接调用 FFI
- `keepass_core` 保持 UI 无关，易于测试
- `keepass_ffi` 是 `keepass_core` 的轻量 C ABI 适配层
- 保存操作是显式的——修改会标记脏状态但不会自动持久化
- 跨边界数据传输使用 JSON DTO
